Ameaça Crítica: FAUX#ELEVATE

Relatório de Inteligência: Mineração e Exfiltração de Dados

Hackers estão utilizando uma nova abordagem para roubar credenciais empresariais e implementando o Crypto miner (Mecanismo de Mineração de Crypto ativos).

Há uma campanha rodando com o foco em ambientes corporativos na qual currículos falsos implantados com um mecanismo de mineradores e roubo de informações utilizando VBScript com ofuscação disfarçados de Doc de currículos/Cv. Os cibercriminosos utilizam E-mails de Phishing; quando ativo, o malware aplica uma execução de ferramentas multiuso configurada para extrair dados e minerar crypto ativos da rede Monero.

O mecanismo se tornou popularmente chamado de Faux#Elevate. A campanha se destaca por seus abusos de infraestruturas legítimas, utilizando DropBox para payloads e sites WordPress comprometidos para controle (C2). Também utilizam SMTP .ru para exfiltração de credenciais de navegadores e arquivos de Desktop.

Os invasores utilizam um sistema de Visual Basic Script que exibe uma mensagem de erro falsa (isca), enquanto um script de 224 mil linhas executa verificações para evitar sandboxes. O malware utiliza Windows Management Instrumentation (WMI) para persistência em sistemas empresariais, desativando o Microsoft Defender e o UAC via registro do Windows.

mozilla.vbs: Roubo de perfil e credenciais do Firefox.
walls.vbs: Exfiltração de arquivos do desktop.
mservice.exe: Minerador XMRig via site WordPress comprometido.
WinRing0x64.sys: Driver de kernel para desbloquear potencial da CPU.
RuntimeHost.exe: Trojan persistente que modifica o Firewall.

"O que torna esta campanha particularmente perigosa é a velocidade de execução: a cadeia completa ocorre em aproximadamente 25 segundos." - Securonix.
