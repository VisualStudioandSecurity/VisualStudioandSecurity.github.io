Campanha FAUX#ELEVATE: O Ataque de 25 Segundos

Publicado por Equipe Técnica Visual Studio & Security | 24 de Março, 2026

A campanha FAUX#ELEVATE utiliza currículos falsos como "bait" para iniciar uma cadeia de infecção extremamente rápida em ambientes corporativos. O malware utiliza scripts VBScript de 224 mil linhas para ocultar sua carga útil real e evitar detecção por sandboxes.

Exfiltração e Mineração

  • Roubo de Dados: Utiliza o projeto ChromElevator para extrair credenciais de navegadores baseados em Chromium.
  • Criptomineração: Instala o minerador mservice.exe (XMRig) utilizando o driver de kernel WinRing0x64.sys para sequestrar o poder da CPU.
  • Destino: Os dados são enviados para contas russas como olga.aitsaid@mail.ru via SMTP.
# Script de Proteção Sugerido
Set-ItemProperty -Path "HKLM:\SOFTWARE\Microsoft\Windows Script Host\Settings" -Name "Enabled" -Value 0
Stop-Process -Name "mservice" -Force -ErrorAction SilentlyContinue

Apple Lança Melhorias de Segurança em Segundo Plano para Corrigir Falha no WebKit

Publicado por Equipe Técnica Visual Studio & Security | 24 de Março, 2026

A Apple iniciou a distribuição de uma nova categoria de atualizações chamada Melhorias de Segurança em Segundo Plano. Esta iniciativa visa reforçar a proteção dos sistemas iOS, iPadOS e macOS sem a necessidade de instalar versões completas de software.

Correção do CVE-2026-20643

A primeira leva dessas atualizações corrige uma vulnerabilidade crítica no WebKit (mecanismo do Safari). O problema envolvia uma brecha de origem cruzada na API de Navegação, que poderia permitir que sites maliciosos acessassem dados de outros domínios, violando o isolamento de segurança.

A falha afetava as versões 26.3.1 do iOS e iPadOS, além do macOS 26.3.1 e 26.3.2. A correção foi implementada através de uma melhoria na validação de entrada, graças ao relato do pesquisador Thomas Espach.

Gestão e Recomendações

As Melhorias de Segurança em Segundo Plano podem ser gerenciadas no menu Privacidade e Segurança das configurações do dispositivo. A Apple recomenda manter a "Instalação Automática" ativada para garantir proteção imediata contra ameaças emergentes, como a recente falha zero-day (CVE-2026-20700) corrigida anteriormente.
